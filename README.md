# Koukan Email MTA

Koukan is a clean-sheet MTA implementation built with the goal of
integrating more easily into modern applications and infrastructure. 

Koukan is written in Python3 leveraging many high-quality
python implementations of email standards that already exist such as smtplib,
aiosmtpd, email.EmailMessage, and dkimpy.
